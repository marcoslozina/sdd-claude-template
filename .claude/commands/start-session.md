Iniciá una nueva sesión de desarrollo siguiendo este protocolo exacto:

1. Hacé las 4 preguntas de setup definidas en CLAUDE.md (lenguaje, tipo de sistema, prototipo/producción, restricciones)

2. Esperá las respuestas del usuario. No continúes sin ellas.

3. Con las respuestas:
   - Confirmá qué skills vas a cargar (lenguaje + roles relevantes)
   - Leé cada skill mencionado desde `.claude/skills/`
   - Confirmá con el usuario: "Listo. Stack: [X]. Roles activos: [Y, Z]. ¿Arrancamos con el problema?"

4. Una vez confirmado, pedí la descripción del problema y ejecutá la Fase 1 (exploración SDD).
